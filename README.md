# vue로 만든 UI COMPONENT

## 프로젝트 준비
```
npm install
```

### 테스트 실행
```
npm run serve
```

### UI 콤포넌트
* button
  * CommonButton
  * CommonSwitchButton
* form
  * CommonForm
  * CommonFormItem
* input
  * CommonInput
* layout
  * CommonCol
  * CommonRow
* radio
  * CommonRadio
  * CommonRadioGroup
* select
  * CommonSelect
  * CommonSelectOption
